# Github User App
This app is for Submission Belajar Fundamental Aplikasi Android on Dicoding, use this only for references.

## Features
- Built on Kotlin
- Light/Dark Mode
- Using Github API
- Implement Material 3
- Implement Android Architecture Component
    - LiveData & ViewModel
    - Datastore
    - Room
